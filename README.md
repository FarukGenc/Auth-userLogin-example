# Sign-in & Sign-up App. with ReactJs/Firebase!

  - React Router 4
  - Firebase
  - Boostrap


## Installation

----

Step by step
```sh
$ git clone https://github.com/FarukGenc/Auth-userLogin-example.git
```

```sh
$ cd Auth-userLogin-example
```

```sh
$ npm start
```

Look: 
```sh
localhost:3000
```

That's All ! Hepsi bu kadar...
